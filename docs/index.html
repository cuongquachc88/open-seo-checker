Features

Everything an SEO audit needs, nothing it doesn't.

Each feature is built to produce exportable, evidence-based findings — not vague scores. Drives a self-hosted React SPA and a Hono JSON API you can script against.

Crawl

JS-aware crawling via Playwright Chromium. Respects robots.txt, follows redirects, captures response codes, headers, and rendered HTML.

Audit issues

Title, meta, headings, canonical, hreflang, indexability, schema.org, OpenGraph, image alts, broken anchors, redirect chains, and 30+ other rules. Each with a counter-example.

Sitemaps

Generate XML sitemaps directly from a crawl. Detect orphan URLs not referenced from any internal link. Drifts against the expected sitemap are flagged per-URL.

Compare runs

Diff two crawls side by side: new URLs, lost URLs, regressions in titles / canonicals / status codes. Saves the developer from "did this PR break ten pages?" guesswork.

AI insights

Optional pluggable AI providers summarise the issues list into actionable briefs. Bring your own key (OpenAI, Anthropic, Gemini, Ollama, …) — never proxied through us.

Export

CSV, JSON, or XLSX. Pipe straight into a job pipeline, drop into Sheets, or archive for trend-tracking over time. Schema is documented and stable.

Workflow

Three steps from install to insight.

The whole loop runs on your laptop. No proxy, no queue, no third-party data store. Plenty fast enough for sites with a few thousand pages.

Install

One shell command on macOS / Linux (or one batch file on Windows). The installer drops a real desktop shortcut that launches the server + opens the dashboard.

Crawl

Drop a URL into the dashboard. The background worker renders JS, follows redirects, stores each page in a local SQLite database, and streams progress to the UI live.

Audit & export

Filter the issue list, drill into a URL, run a sitemap diff, compare two runs. Hit Export when you're ready; pick CSV, JSON, or XLSX; download starts instantly.

Install

It takes one line.

The script downloads the latest self-contained bundle from GitHub Releases, extracts it to a temp directory, runs the platform-specific installer, and cleans up after itself. Source builds are also supported if you prefer to compile yourself.

One-liner (No clone required)

Downloads the latest macOS bundle, extracts, runs the installer. The installer drops an .app shortcut on your Desktop — double-click it any time to start the server.

$ OSE_OWNER=cuongquachc88 \
   curl -fsSL https://raw.githubusercontent.com/cuongquachc88/open-seo-checker/main/scripts/install-from-release.sh | bash
✓ Open SEO Checker installed. Double-click the desktop shortcut to start.

Build from source

Requires Node 18+ and pnpm. Useful for contributing or for hosts without an official bundle (FreeBSD, headless Linux without GUI, etc.).

$ git clone https://github.com/cuongquachc88/open-seo-checker
$ cd open-seo-checker
$ pnpm install    # installs deps + Playwright Chromium
$ pnpm build
$ pnpm server    # http://localhost:7437

Who it's for

Three personas, same toolbox.

Open SEO Checker was built for technical marketers, agencies, and the platform engineers who serve both. The same data drives three workflows.

In-house Solo technical SEO Run nightly crawls against your own properties, diff against last week, ship regressions as PR-blocking checks.
Agency Client deliverables Spin up a fresh crawl per project, export a CSV/XLSX report into the deck, throw away the workspace when the engagement ends. Nothing about the client leaks.
Platform team CI / dev infra Run the API + CLI against a staging URL in CI; gate releases on regressions in canonical / status / meta. Auth via API key.
