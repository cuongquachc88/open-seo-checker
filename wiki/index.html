Quick reference

Most-used commands. The full table lives in README.md at the project root and is repeated inside each guide. 

$ pnpm install           # deps + Playwright Chromium
$ pnpm server            # production bootstrap (single port :7437, blue BE banner)
$ pnpm start:sh         # alias: pnpm dev:sh — orchestrator + Node/pnpm auto-install
$ pnpm dev:web           # Vite dev server (magenta FE banner)
$ pnpm monitor           # second-screen tailer
$ pnpm test:unit         # 26 vitest specs
$ pnpm test:e2e          # 23 Playwright specs
$ pnpm crawl <url>       # one-off CLI crawl (CSV/JSON/XLSX export)
